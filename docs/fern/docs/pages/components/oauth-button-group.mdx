# OAuthButtonGroup

Renders OAuth buttons for all enabled providers on the project dashboard.

## Props

- `type`: `'sign-in' | 'sign-up'` - Specifies whether the buttons are for sign-in or sign-up (functionality remains the same).
- `mockProject` (optional): `{ config: { oauthProviders: { id: string }[] } }` - Used for mocking project configuration in testing scenarios.

## Example

```tsx
import { OAuthButtonGroup } from '@stackframe/stack';

export default function Page() {
  return (
    <div>
      <h1>Sign In</h1>
      <OAuthButtonGroup type='sign-in' />
    </div>
  );
}
```
