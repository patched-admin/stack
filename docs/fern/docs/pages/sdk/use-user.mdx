# useUser

A custom hook that fetches and manages user data.

## Parameters

This hook does not take any parameters.

## Returns

An object with the following properties:

- `user`: `User | undefined` - The user data if available.
- `isLoading`: `boolean` - Indicates whether the data is currently being fetched.
- `isError`: `boolean` - Indicates whether an error occurred during the fetch.

## Example

```tsx
import { useUser } from '@stackframe/stack';

function UserProfile() {
  const { user, isLoading, isError } = useUser();

  if (isLoading) return <div>Loading...</div>;
  if (isError) return <div>Error fetching user data</div>;
  if (!user) return <div>No user data available</div>;

  return <div>Welcome, {user.name}!</div>;
}
```
