# MagicLinkSignIn

A component that handles the magic link sign-in process, allowing users to sign in using their email address.

## Props

This component does not accept any props.

## Example

```tsx
import { MagicLinkSignIn } from '@stackframe/stack';

function SignInPage() {
  return (
    <div>
      <h1>Sign In</h1>
      <MagicLinkSignIn />
    </div>
  );
}
```

This component handles the entire magic link sign-in flow, including email input, sending the magic link, and OTP verification.
