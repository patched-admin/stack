# PasswordReset

A component that handles the password reset process, including validation of the reset link and rendering appropriate messages.

## Props

- `searchParams`: `Record<string, string>` - An object containing search parameters, expected to include a `code` for password reset.
- `fullPage` (optional): `boolean` - Determines if the component should render in full-page mode. Defaults to `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

function ResetPasswordPage() {
  const searchParams = {
    code: 'reset-code-from-url',
  };

  return (
    <PasswordReset
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```

This component will handle different scenarios such as invalid, expired, or already used reset links, and render the appropriate message or the password reset form.
