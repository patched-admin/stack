# EmailVerification

A component that handles the email verification process.

## Props

- `searchParams` (optional): `Record<string, string>` - An object containing search parameters, typically including a verification code.
- `fullPage` (optional): `boolean` - Determines whether the component should be displayed in full-page mode.

## Example

```tsx
import { EmailVerification } from '@stackframe/stack';

function VerificationPage() {
  const searchParams = {
    code: 'verification_code_here'
  };

  return (
    <EmailVerification
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```

This component handles various states of the email verification process, including initial verification prompt, success message, and error cases such as invalid or expired verification links.
