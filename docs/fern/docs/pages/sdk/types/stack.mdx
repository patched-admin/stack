# StackProvider

A component that provides Stack context to its children, including app configuration, language settings, and translations.

## Props

- `children`: `React.ReactNode` - The child components to render within the StackProvider.
- `app`: `StackClientApp<true> | StackServerApp<true> | StackAdminApp<true>` - The Stack app instance to use for configuration.
- `lang` (optional): `React.ComponentProps<typeof TranslationProvider>['lang']` - The language to use for translations.
- `translationOverrides` (optional): `Record<string, string>` - A mapping of English translations to translated equivalents. These take priority over the translations from the specified language.

## Example

```tsx
import { StackProvider, StackClientApp } from '@stackframe/stack';

const app = new StackClientApp({
  projectId: 'your-project-id',
  // other configuration options
});

function App() {
  return (
    <StackProvider app={app} lang='en'>
      {/* Your app components */}
    </StackProvider>
  );
}
```
