# EmailVerification

A component that handles email verification process.

## Props

- `searchParams` (optional): `Record<string, string>` - Search parameters containing the verification code.
- `fullPage` (optional): `boolean` - Whether to display the component in full-page mode.

## Example

```tsx
import { EmailVerification } from '@stackframe/stack';

function EmailVerificationPage() {
  const searchParams = { code: 'verification_code_here' };

  return (
    <EmailVerification
      searchParams={searchParams}
      fullPage={true}
    />
  );
}
```
