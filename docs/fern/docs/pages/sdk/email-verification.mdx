# EmailVerification

A component that handles email verification process and displays appropriate messages.

## Props

- `searchParams`: `Record<string, string>` (optional) - An object containing search parameters. It should include a `code` property for the verification code.
- `fullPage`: `boolean` (optional) - Determines if the component should render in full-page mode. Default is `false`.

## Example

```tsx
import { EmailVerification } from '@stackframe/stack';

// Basic usage
<EmailVerification />

// With search params and full page mode
const searchParams = { code: 'abc123' };
<EmailVerification searchParams={searchParams} fullPage={true} />

// Typical usage in a Next.js page component
export default function VerificationPage({ searchParams }) {
  return <EmailVerification searchParams={searchParams} />;
}
```
