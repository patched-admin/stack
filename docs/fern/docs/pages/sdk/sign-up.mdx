# SignUp

Renders a sign-up page component, which can be displayed as a full page or embedded.

## Props

- `fullPage` (optional): `boolean` - Determines if the sign-up page should be displayed as a full page. Default is `false`.

## Example

```tsx
import { SignUp } from '@stackframe/stack';

// Embedded sign-up component
const EmbeddedSignUp = () => {
  return <SignUp />;
};

// Full-page sign-up component
const FullPageSignUp = () => {
  return <SignUp fullPage={true} />;
};
```
