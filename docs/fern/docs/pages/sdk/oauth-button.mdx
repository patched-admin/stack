# OAuthButton

A customizable button component for OAuth authentication with various providers.

## Props

- `provider`: `string` - The name of the OAuth provider (e.g., 'google', 'github', 'facebook', etc.).
- `type`: `'sign-in' | 'sign-up'` - Specifies whether the button is for signing in or signing up.

## Example

```tsx
import { OAuthButton } from '@stackframe/stack';

function AuthPage() {
  return (
    <div>
      <OAuthButton provider="google" type="sign-in" />
      <OAuthButton provider="github" type="sign-up" />
    </div>
  );
}
```
