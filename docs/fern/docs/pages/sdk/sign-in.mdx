# SignIn

Renders a sign-in page component with customizable options.

## Props

- `fullPage` (optional): `boolean` - If true, renders the sign-in page in full-page mode.
- `automaticRedirect` (optional): `boolean` - If true, enables automatic redirection after successful sign-in.
- `extraInfo` (optional): `React.ReactNode` - Additional information to display on the sign-in page.
- `firstTab` (optional): `'magic-link' | 'password'` - Determines which tab is displayed first on the sign-in page.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

function MySignInPage() {
  return (
    <SignIn
      fullPage={true}
      automaticRedirect={true}
      extraInfo={<p>Welcome back!</p>}
      firstTab="magic-link"
    />
  );
}
```
