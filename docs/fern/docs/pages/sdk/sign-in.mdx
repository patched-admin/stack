# SignIn

Renders a sign-in page component.

## Props

- `fullPage` (optional): `boolean` - Determines if the sign-in page should be displayed in full-page mode. Default is `false`.

## Example

```tsx
import { SignIn } from '@stackframe/stack';

// Default usage (not full page)
const SignInComponent = () => <SignIn />;

// Full page usage
const FullPageSignInComponent = () => <SignIn fullPage={true} />;
```
