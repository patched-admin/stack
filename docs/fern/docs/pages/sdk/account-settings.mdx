# AccountSettings

Renders the account settings page with various sections such as profile, security, and team management.

## Props

- `fullPage` (optional): `boolean` - Determines if the component should be rendered in full-page mode. Default is `false`.

## Example

```tsx
import { AccountSettings } from '@stackframe/stack';

function MyAccountPage() {
  return (
    <AccountSettings fullPage={true} />
  );
}

// Or with default fullPage value
function MyDefaultAccountPage() {
  return (
    <AccountSettings />
  );
}
```
