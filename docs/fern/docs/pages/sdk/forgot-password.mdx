# ForgotPassword

A component that renders a forgot password form with optional full-page layout.

## Props

- `fullPage` (optional): `boolean` - Determines if the component should be rendered in full-page mode. Default is `false`.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Default usage
const DefaultForgotPassword = () => <ForgotPassword />;

// Full-page layout
const FullPageForgotPassword = () => <ForgotPassword fullPage={true} />;
```
