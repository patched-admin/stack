# ForgotPassword

A React component that renders a forgot password form or relevant messages based on the user's state.

## Props

- `fullPage` (optional): `boolean` - Determines whether the component should be displayed in full-page mode.

## Example

```tsx
import { ForgotPassword } from '@stackframe/stack';

// Basic usage
<ForgotPassword />

// Full page mode
<ForgotPassword fullPage={true} />
```
