# PasswordReset

A React component for handling password reset functionality.

## Props

- `searchParams`: `Record<string, string>` - An object containing search parameters, typically including a 'code' for password reset.
- `fullPage` (optional): `boolean` - Determines if the component should render in full-page mode. Defaults to `false`.

## Example

```tsx
import { PasswordReset } from '@stackframe/stack';

function PasswordResetPage() {
  const searchParams = { code: 'reset123' };
  
  return (
    <PasswordReset 
      searchParams={searchParams}
      fullPage={true}
    />
  );
}

// Usage without fullPage prop
function PasswordResetInline() {
  const searchParams = { code: 'reset456' };
  
  return <PasswordReset searchParams={searchParams} />;
}
```
